# Introducción al canal web {#get-started-web}

>[!BEGINSHADEBOX]

**En esta página:** empiece a usar el canal web en Adobe Journey Optimizer para crear y ofrecer experiencias web personalizadas en los recorridos y campañas de los clientes.

>[!ENDSHADEBOX]

[!DNL Journey Optimizer] le permite crear y enviar visualmente experiencias web personalizadas a sus clientes.

Mediante una interfaz visual intuitiva, utilice el canal web para modificar sus propiedades web fácilmente para experimentar, optimizar y personalizar sus campañas de usuario final.

![](../rn/assets/do-not-localize/web-authoring.gif)

Si ya está utilizando canales de salida para el envío de mensajes como el correo electrónico, SMS o notificaciones push, ahora puede aprovechar el canal web de entrada para ofrecer una experiencia verdaderamente personalizada que abarque todos los canales.

Una vez creado un recorrido o una campaña, seleccione **Web** como acción y defina la configuración básica. Para obtener más información sobre cómo configurar el recorrido o la campaña, consulte esta [página](create-web.md#create-web-experience).

>[!NOTE]
>
>Si es la primera vez que crea una experiencia web, asegúrese de seguir los requisitos previos descritos en [esta sección](web-prerequisites.md).

Descubra los pasos detallados para crear una campaña web en [este vídeo](create-web.md#video).

## Recursos adicionales

* **[Cree experiencias web](create-web.md)**: aprenda a crear y configurar campañas web y recorridos para modificar el contenido web.
* **[Requisitos previos del canal web](web-prerequisites.md)**: conozca los requisitos técnicos y la configuración necesarios para la implementación del canal web.
* **[Edite contenido web](create-web.md#edit-web-content)**: domine el diseñador web para modificar páginas utilizando modos de edición visuales o no visuales.
* **[Administre modificaciones web](manage-web-modifications.md)**: aprenda a organizar, aplicar y administrar modificaciones en las experiencias web.
* **[Monitorice experiencias web](monitor-web-experiences.md)**: haga un seguimiento y analice el rendimiento de sus campañas web con informes detallados.
* **[Genere contenido web con el Asistente de IA](../content-management/generative-full-content.md)**: aproveche la IA para crear y optimizar contenido web con texto e imágenes.
* **[Tutoriales de campañas web](https://experienceleague.adobe.com/es/docs/journey-optimizer-learn/tutorials/channels/web-channel/create-a-web-campaign){target="_blank"}**: explore tutoriales de vídeo paso a paso sobre las características del canal web y las prácticas recomendadas.
