# Actividad Finalizar{#end-activity}

>[!BEGINSHADEBOX]

**En esta página:** Aprenda a utilizar la actividad Fin para marcar el final de cada ruta de recorrido y agregar etiquetas que faciliten la lectura de los informes de recorrido.

>[!ENDSHADEBOX]

La actividad **[!UICONTROL End]** le permite marcar el final de cada ruta de acceso del recorrido. No es obligatorio, pero se recomienda para la claridad visual. De hecho, si el recorrido tiene varias actividades finales, le recomendamos que agregue una etiqueta en cada final para facilitar la lectura de los informes. * **TL;DR:** En esta página se explica la actividad Fin en Journey Optimizer, un marcador visual colocado al final de cada ruta de acceso de recorrido que es opcional pero recomendado para la legibilidad del informe.

**Intenciones:**

* Añada una actividad End para marcar la finalización de una ruta de recorrido
* Etiquetar actividades de finalización para mejorar la legibilidad del informe de recorrido
* Distinguir visualmente varias rutas de recorrido mediante nodos finales etiquetados

**Glosario:**

* **Actividad final**: Un nodo de recorrido que marca visualmente el final de una ruta de recorrido; no es obligatorio, pero ayuda a la claridad *(específico del producto)*

**Protecciones:**

* La actividad final no es obligatoria; los recorridos funcionan sin ella.
* Se recomienda añadir una etiqueta a cada actividad End cuando un recorrido tiene varias rutas, para que los informes en directo sean más fáciles de interpretar.

**Terminología:**

* Nombre canónico: Actividad final — Acrónimo: n/a — variantes: Nodo final, Etiqueta final
* No confunda: &quot;End activity&quot; ≠ &quot;End tag&quot;: la etiqueta End es un nodo generado automáticamente visible durante la creación del recorrido, mientras que la actividad End es un componente colocable

**PREGUNTAS MÁS FRECUENTES:**

* **Q: ¿Se requiere la actividad End para publicar un recorrido?** — No, no es obligatorio, pero se recomienda para una claridad visual.
* **Q: ¿Por qué debería etiquetar actividades End?** — Cuando un recorrido tiene varias rutas, las etiquetas ayudan a distinguirlas en los informes de recorrido.
* **Q: ¿Dónde puedo ver el impacto de las etiquetas de actividad End?** — Las etiquetas aparecen en el informe en vivo de recorrido, lo que facilita el seguimiento del rendimiento a nivel de ruta.
