# currentTimeInMillis {#currentTimeInMillis}

## Sintaxis de función

`currentTimeInMillis()`

## Parámetros

Esta función no utiliza parámetros.

## Firmas y tipo devuelto

`currentTimeInMillis()`

Devuelve un número entero.

## Ejemplos
