# isEmpty {#isEmpty}

Devuelve true si la cadena del parámetro no tiene ningún carácter.

## Categoría

Cadena

## Sintaxis de función

`isEmpty(<parameters>)`

## Parámetros

* string

## Firma y tipo devuelto

`isEmpty(<string>)`

Devuelve un valor booleano.

## Ejemplo

`isEmpty("")`

Devuelve verdadero.

`isEmpty("Hello World")`

Devuelve falso.
