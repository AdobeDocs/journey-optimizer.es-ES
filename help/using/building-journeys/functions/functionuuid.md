# uuid {#uuid}

Genera un UUID aleatorio (identificador único universal).

## Categoría

Cadena

## Sintaxis de función

`uuid()`

## Parámetros

Esta función no requiere parámetros.

## Firma y tipo devuelto

`uuid()`

Devuelve una cadena.

## Ejemplo

`uuid()`

Devuelve &quot;79e70b7f-8a85-400b-97a1-9f9826121553&quot;.s
