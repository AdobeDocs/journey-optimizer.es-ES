# contain {#contain}

Comprueba si la cadena del segundo argumento está contenida en la primera.

## Categoría

Cadena

## Sintaxis de función

`contain(<parameters>)`

## Parámetros

* cadena

## Firma y tipo devuelto

`contain(<string>,<string>)`

Devuelve un valor booleano.

## Ejemplo

`contain("rowing is great", "great")`

Devuelve verdadero.
