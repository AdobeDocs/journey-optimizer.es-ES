# isNotEmpty {#isNotEmpty}

Devuelve true si la cadena del parámetro no está vacía.

## Categoría

Cadena

## Sintaxis de función

`isNotEmpty(<parameters>)`

## Parámetros

* string

## Firma y tipo devuelto

`isNotEmpty(<string>)`

Devuelve un valor booleano.

## Ejemplo

`isNotEmpty("")`

Devuelve falso.

`isNotEmpty("hello")`

Devuelve verdadero.
