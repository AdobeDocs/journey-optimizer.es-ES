# Permisos integrados {#ootb-permissions}

Los permisos integrados representan los diferentes permisos que se pueden asignar a una **[!UICONTROL función]** para ajustar el acceso de los usuarios a Journey Optimizer. Los permisos de alto nivel abarcan permisos de bajo nivel, detallados en [esta página](high-low-permissions.md).

| Capacidad | Permisos |
|-|-|
| Asistente de IA | **[!DNL Generate content]**: acceso al Asistente de IA en Journey Optimizer. |
| Campañas | **[!DNL Approve & publish Campaigns]**: derecho para aprobar y publicar campañas cuando se aplique una directiva. </br>**[!DNL Manage campaigns]**: leer, crear, editar y eliminar campañas.</br>**[!DNL Publish campaigns]**: derecho para publicar campañas.</br>**[!DNL View campaigns]**: acceso de solo lectura a las campañas.</br>**[!DNL View campaigns report]**: leer y editar informes de campaña. |
